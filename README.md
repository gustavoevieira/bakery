This is a **fictional** website project developed to demonstrate my **front-end** web development, design, and UI creation skills. It simulates the online presence of a handcrafted bakery, focusing on an innovative, vibrant, and inviting design.

---

## ✨ Project Overview

The "Doce Sabor" website is a virtual showcase for a boutique bakery, designed to attract customers through an appetizing visual identity and a smooth navigation experience. The primary goal was to create a visually rich design that conveys the sweetness of the products and the warmth of the environment.

---

## 🚀 Technologies Used

* **HTML5**: Semantic structuring of page content.
* **CSS3**: Complete layout styling, focusing on:
    * A vibrant and harmonious color palette inspired by sweets.
    * Responsive design for various screens (mobile, tablet, desktop).
    * Use of CSS Variables for easy maintenance of the theme.
* **JavaScript**: Added interactivity and subtle visual effects:
    * **Intersection Observer API**: For "Scroll Reveal" animations (elements fading in as you scroll).
    * **Parallax Effect**: On the Hero Section.
    * **Smooth Scrolling**: Between navigation sections.
    * **Custom Lightbox/Carousel**: A modal system for the image gallery.
* **Font Awesome**: For modern and scalable icons.
* **Google Fonts**: Engaging typography (`Dancing Script` for titles and `Open Sans` for body text).
* **Git & GitHub**: Version control and source code hosting.

---

## ✨ Features

The "Doce Sabor" website features the following sections and characteristics:

* **Header & Navigation:** A modern header with a logo and intuitive navigation links.
* **Hero Section:** A high-impact landing area with an inviting message and parallax effect.
* **About Us:** Details regarding the history and passion behind the bakery.
* **Our Menu:** Showcasing main products with photos, descriptions, and prices in an attractive grid layout.
* **Image Gallery ("Sweet Moments"):**
    * A grid of high-quality thumbnails of the sweets and the bakery's atmosphere.
    * **Lightbox/Carousel System:** Clicking a thumbnail opens a modal window displaying the enlarged image and allowing navigation through the gallery.
* **Contact Section:** Contact information, business hours, and a static contact form.
* **Footer:** Copyright information and social media links.
* **Animations & Interactivity:** Smooth hover transitions and scroll-triggered animations for a dynamic experience.

## 🛠️ How to Run the Project Locally

This is a **static** project and is very easy to set up!

### Prerequisites
* A modern web browser (Chrome, Firefox, Edge, Safari, etc.).

### Setup
1. **Clone the Repository:**
    ```bash
    git clone [https://github.com/gustavoevieira/doceria-doce-sabor.git](https://github.com/gustavoevieira/doceria-doce-sabor.git)
    ```
    Navigate to the project folder:
    ```bash
    cd doceria-doce-sabor
    ```

2. **Open in Browser:**
    * You can simply open the `index.html` file directly in your browser.
    * **Alternative (Recommended):** Use the "Live Server" extension in VS Code. This ensures all resource paths (CSS, JS, Images) load correctly and provides a better development experience.
